STEP FOR JAVA BEGINNER
----------------------
1. Download JDK dari oracle website
2. Install  JDK -> C:\Program Files\java\jdk..
3. Set Path in Environment Variable (Setting) 
   -> C:\Program Files\java\jdk..\bin
4. Run in GitBash 
   -> javac <nama file>.java
   -> java <nama file>
